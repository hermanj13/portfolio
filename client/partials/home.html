Platos

Connecting people with similar schedules and hobbies, making it easier to maintain social engagements and commitments. Using optimized SQL queries this application is able to rapidly compile matches based on location with the Google Maps API and user inputted schedules. Implemented MaterializeCSS as a front-end CSS framework.

Adoptagram

A central place for animal shelters to post and keep upto date information about animals at their shelter. With an easy to use front and backend, shelters are able to post images and descriptions of animals, while potential animal adopters have a central location to submit applications for adoptions and fosters to multiple agencies. Built using Ruby on Rails for project at The Coding Dojo.
